# UI 요소 기술 명세서 (UI Element Technical Specification)

## 1. 기본 정보
| 항목 | 내용 |
|---|---|
| UI 요소명 | (예: 메인 메뉴 버튼) |
| UI ID | (예: BTN_MAIN_START) |
| UI 유형 | 버튼 / 텍스트 / 이미지 / 슬라이더 / 체크박스 / 팝업 등 |
| 화면/컨텍스트 | (예: 로비 화면, 인게임 HUD) |

---

## 2. 목적 및 설명
- 목적: 해당 UI 요소가 제공해야 하는 핵심 기능
- 설명: 사용자 관점에서의 역할 요약  
  (예: “게임 시작을 트리거하는 메인 인터랙션 버튼”)

---

## 3. 시각적 사양 (Visual Spec)
| 항목 | 내용 |
|---|---|
| 위치 | 기준 앵커, 좌표, 레이아웃 규칙 |
| 크기 | Width × Height (px / 비율) |
| 색상 | 기본 / Hover / Disabled |
| 텍스트 | 폰트, 크기, 자간, 정렬 |
| 애니메이션 | 등장 / 강조 / 클릭 시 |
